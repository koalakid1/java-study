이미지포맷

1. bitmap
=> .bmp
=> 기본적인 이미지 포맷

2.gif
=> .gif
=> 웹에 사용하기 위해서 만듬. 작은 아이콘. 저해상도용

3. jpg
=> .jpg or .jpeg ( 현대는 .jpg로 거의 정착)
=> 웹에서 사용하기 위해 만듬. 사진. 고해상도용

4. png
=> .png
=> gif와 jpg의 장점을 살려서 만듬.
=> 현재 web, app 에서 표준으로 사용
